# Decision Log: [Project Name]

## Purpose
Track significant decisions with rationale.
This prevents re-litigating settled questions.

## Format
Each decision should include:
- Date
- Decision
- Rationale
- Alternatives considered
- Who/what decided (human, AI tool, agent)
